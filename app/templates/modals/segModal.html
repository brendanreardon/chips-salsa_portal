Optional Field: Copy Number Alterations

The current version of The Molecular Oncology Almanac can process total copy number segment files, which we recommend are generated from GATK CNV and Oncotator. The Molecular Oncology Almanac specifically requires the following columns:

gene, segment_contig, segment_start, segment_end, sample, segment_mean

Example upload file for germline variants using required columns:
gene segment_contig segment_start segment_end sample segment_mean
